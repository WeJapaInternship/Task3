
# To get started, run the following commands:
1 npm install <br/>
2 npm start

the following shows how to perform the crud operations:<br/><br/>

make a POST request with keys *name* and *word* to *127.0.0.1:3030/add* to add a new note, returns a json response<br/>
make a GET request to *127.0.0.1:3030/get* to get all notes, returns a json response<br/>
make a POST request with keys *nameid* and *word* to *127.0.0.1:3030/update* to update a note, returns a json response<br/>
make a GET request with query *nameid* to *127.0.0.1:3030/delete* to delete a note, returns a json response<br/>
